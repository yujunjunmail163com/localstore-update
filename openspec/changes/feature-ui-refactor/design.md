## Context

当前扩展使用 Material Design 3 深色主题（#1e1f20 背景，#8ab4f8 强调色）。需要全面切换为白色商务风格，以匹配企业级工具的视觉标准。

## Goals / Non-Goals

**Goals:**
- 实现白色商务风格配色系统
- 按钮固定尺寸、4px 圆角、精确的 hover 效果
- 搜索框收窄圆角至 4px
- 表格使用 #e7ebf3 表头 + #ebeef5 分隔线
- 删除按钮改为轻量文字链接风格
- 编辑态使用淡蓝阴影或单像素边框

**Non-Goals:**
- 不改变功能逻辑
- 不改变整体布局结构
- 不引入外部依赖

## Decisions

### 1. 配色方案

| 角色 | 颜色值 |
|---|---|
| 主背景 | #ffffff |
| 表头背景 | #e7ebf3 |
| 表头文字 | #637598 |
| 表格分隔线 | #ebeef5 |
| 主文字 | #333333 |
| 次文字 | #637598 |
| 主按钮 | #003AA8 |
| 主按钮 hover | #3361b9 |
| 次按钮边框 | #C7D5EB |
| 删除按钮文字 | #637598 |
| 编辑态焦点边框 | #C0D0E8 |
| 编辑态焦点阴影 | rgba(0,58,168,0.08) |

### 2. 按钮规格

所有顶部按钮统一：width: 74px; height: 32px; border-radius: 4px; font-size: 14px。

### 3. 编辑态设计

编辑输入框使用 1px solid #C0D0E8 单边框 + box-shadow: 0 0 0 2px rgba(0,58,168,0.08) 淡蓝阴影，不使用双边框。
